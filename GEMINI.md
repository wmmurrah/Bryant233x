# GEMINI.md: Bryant 233x Boat Management

## Directory Overview

This directory appears to be for the personal management and documentation of a 2013 Bryant 233x boat. It contains checklists for operation, official documentation like bills of sale, insurance and maintenance records, and pictures. The owner's name appears to be Hank Murrah, and the boat is primarily used on Lake Martin.

## Context Knowledge

*   **`Bryant233MarineAssistant.md`**: This is the primary knowledge base file for this project. It contains detailed operational, maintenance, and system-specific information for the Bryant 233x boat. This file should be used as the main context for any AI assistant interactions.

## Key Files & Directories

*   **`checklists/`**: This directory contains Quarto (`.qmd`) source files and their rendered PDF versions for various boat operations.
    *   `bryant-staging-launch-checklist.qmd`: A detailed checklist for preparing the boat for launch at the ramp.
    *   `bryant-trailering-checklist.qmd`: A checklist for safely trailering the boat with a Ford F-150 and a 2016 Magic Tilt Tandem trailer.
    *   `safety-briefing.qmd`: A script for the captain's pre-launch safety briefing for passengers.

*   **`documentation/`**: Contains official records for the boat and trailer. This includes what appears to be a bill of sale and registration documents.

*   **`insurance/`**: Contains the boat's insurance documents.

*   **`maintenance/`**: Contains maintenance records for the boat. This includes records from "280 Marine" and "Harris Performance".

*   **`pictures/`**: Contains photos of the boat, including pictures of the house and starter batteries.

## Usage

This directory is used as a central repository for all information related to the Bryant 233x boat. The checklists can be used to ensure safe and repeatable operations, and the other directories provide a place to store important records. The `.qmd` files in the `checklists` directory can be edited and re-rendered to produce updated PDF checklists.
